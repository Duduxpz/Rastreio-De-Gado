## 📝 Notas Importantes

1. **Supabase**: Configurar project URL e ANON_KEY em `.env.local`
2. **RLS**: Ativar Row Level Security nas tabelas
3. **Auth**: Configurar Site URL nos settings do Supabase
4. **Deploy**: Adicionar variáveis de ambiente em Vercel
